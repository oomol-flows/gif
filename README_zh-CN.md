# GIF 处理工具包

一套用于处理 GIF 动画的可视化工作流模块集合。这个工具包让您能够通过简单易用的构建模块来操作 GIF 文件，无需任何编程知识。

## 项目简介

本项目提供了一套**可视化工作流模块**，可以连接在一起处理 GIF 动画。就像搭建积木一样 - 每个模块执行特定的任务，您可以将它们链接在一起创建强大的 GIF 处理工作流。

适用于：
- **内容创作者**需要编辑 GIF 动画
- **社交媒体运营**优化不同平台的 GIF
- **数字营销人员**创建引人入胜的动画内容
- **任何人**处理 GIF 文件并需要简单而强大的工具

## 可用模块

### 🔄 **GIF 分割**
*从 GIF 动画中提取单独的帧*

**功能说明：** 将 GIF 文件分解为单独的图像帧
- **输入：** GIF 文件
- **输出：** 多个图像文件（PNG、JPG 或 BMP 格式）
- **使用场景：** 需要编辑特定帧或将 GIF 转换为图像序列

### 📐 **GIF 尺寸调整**
*按尺寸或百分比调整 GIF 动画大小*

**功能说明：** 改变 GIF 动画的尺寸
- **调整选项：** 设置具体宽高、按百分比缩放或保持长宽比
- **质量控制：** 选择不同的重采样方法以获得最佳质量
- **使用场景：** 为不同平台优化 GIF 或减少文件大小

### 🎬 **GIF 合成**
*将多个图像合并为动画 GIF*

**功能说明：** 从一系列图像创建 GIF 动画
- **输入：** 多个图像文件
- **控制选项：** 设置帧持续时间、循环次数和时间
- **使用场景：** 从照片序列或插图创建 GIF

### ↩️ **GIF 反转**
*反转 GIF 帧的播放顺序*

**功能说明：** 让您的 GIF 动画倒放
- **简单操作：** 一键反转
- **保持质量：** 反转过程中不损失质量
- **使用场景：** 创建"来回"效果或艺术性的反向动画

### ⚡ **GIF 速度调节**
*通过 FPS 或倍数调整 GIF 播放速度*

**功能说明：** 让您的 GIF 播放更快或更慢
- **速度控制：** 使用倍数（2倍更快、0.5倍更慢）或设置特定 FPS
- **精确时序：** 微调动画速度以获得完美时机
- **使用场景：** 加速慢过程或减慢快速动作

### ✂️ **GIF 裁剪**
*将 GIF 动画裁剪为指定的矩形区域*

**功能说明：** 从 GIF 中切出矩形部分
- **精确控制：** 设置确切的坐标和尺寸
- **保持动画：** 所有帧都会一致地裁剪
- **使用场景：** 聚焦动画的特定部分或移除不需要的区域

### 🗜️ **GIF 优化**
*通过减少颜色和帧数优化 GIF 文件大小*

**功能说明：** 在保持视觉质量的同时减少文件大小
- **优化级别：** 选择基础、中等或强力压缩
- **颜色减少：** 减少调色板颜色以降低文件大小
- **帧数减少：** 移除帧以减少文件大小和提高速度
- **数据分析：** 查看确切的空间节省量
- **使用场景：** 为网络上传准备 GIF 或满足文件大小限制

## 使用方法

1. **选择模块：** 选择任务所需的处理模块
2. **连接模块：** 按您希望的处理顺序链接模块
3. **配置设置：** 为每个模块调整尺寸、速度、质量等参数
4. **运行工作流：** 自动处理您的 GIF 文件
5. **获取结果：** 接收处理后的 GIF 文件，即可使用

## 常见工作流

### 📱 **社交媒体优化**
`GIF 输入 → 尺寸调整 → 优化 → 输出`
- 调整到平台要求
- 优化快速加载

### 🎨 **创意效果**
`GIF 输入 → 反转 → 速度调节 → 输出`
- 创建来回效果
- 微调时序

### ✂️ **内容编辑**
`GIF 输入 → 裁剪 → 尺寸调整 → 优化 → 输出`
- 聚焦重要区域
- 为不同平台准备

### 🔧 **GIF 分析与重建**
`GIF 输入 → 分割 → (编辑帧) → 合成 → 输出`
- 提取帧进行详细编辑
- 以自定义时序重建

## 支持的文件格式

- **输入：** GIF 文件
- **输出：** GIF 文件、PNG、JPG、BMP（取决于模块）
- **质量：** 除非特别要求优化，否则保持原始质量

- **批量处理：** 高效处理多个文件

- **格式灵活：** 支持多种图像格式
